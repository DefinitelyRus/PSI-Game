Your mission is simple --- enter the datacenter and destroy it.
You're in command of Mira and Orra Kale, two armed civilians volunteering to put a stop to the AI takeover.

Unit controls:
- 1, 2 -> Select unit
- ~ -> Select both
- ESC -> Deselect both
- LMB -> Move and attack any
- RMB -> Move and target on cursor
- Space -> Stop moving

Camera Controls:
- W, A, S, D -> Move camera
- Drag + Middle-click -> Move camera
- Shift + 1, 2 -> Focus on unit

Inventory Controls:
- Z, X, C -> Toggle item power
- Shift + Z, X, C -> Drop item

Others:
- Hold ESC -> Exit game
- [DEBUG] Alt + W + X + F4 -> Auto-win
